# BisacSuggester

Node.js ve TypeScript ile geliştirilmiş, kitap başlığı, içerik, sepet ve kısa açıklama girdilerini alıp BISAC kodu öneren bir REST API sunucusu.

## Kurulum

```bash
npm install
```

## Kullanım

```bash
npm run dev
```

## API

```bash
POST /api/suggest
```

## Test

```bash
npm run test
```
